## Business Logic : 

## If Filter has button : 
1) Main Filter ko child banaounga aur list ko parent main page ka because mujhe filter me state rkhni h aur jab main filter wali button ko click krunga to mujhe parent ki state me jo list ka data pda h usse filter krke dikhana h. 

## If Filter has no button : 
Main MainLayout banounga, Filter aur List ko child banaounga because mujhe filter me state rkhni h aur jab main filter me change event fire krunga to mujhe List ki state me jo data pda h usse filter krke dikhana h.
